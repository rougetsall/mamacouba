== Description ==

Olark is on a mission to humanize business. We make easy-to-use software that helps your team work together to connect with customers. Our live chat product allows your team to begin talking immediately to customers visiting your website. Installation is quick and seamless, and everything you do in Olark integrates directly into many other popular tools that you already use.

A few notes about the sections above:

== Installation ==

1. Go to the 'Plugins' menu in your WordPress instance
2. Select 'Add New'
3. Search for 'Olark Live Chat'
4. Select 'Install' on the 'Olark Live Chat' Plugin by 'Olark'.
*Note:* There are other Olark plugins that are not officially supported by Olark. Make sure you select the correct one.
5. Activate the plugin
6. Navigate to Settings > Olark Settings to add your Site ID and enable the chatbox
